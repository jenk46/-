SCHIZOPHRENIA ESCAPE

STAGE: 1 / 10

[STAGE 1] 암호 해독: 뇌내 도파민 가설
조현병의 주요 가설에 따르면 특정 경로의 도파민 과잉/저활성이 증상을 결정합니다. 아래 기호에 해당하는 경로의 순서 번호를 조합하세요.

1. Nigrostriatal   2. Mesolimbic   3. Mesocortical   4. Tuberoinfundibular

  • (A) 양성 증상(환각, 망상)의 원인 경로
  • (B) 음성 증상 및 인지 저하의 원인 경로
  • (C) 항정신병약 사용 시 프로락틴 상승과 관련된 경로

답: ABC 순서대로 숫자 3자리를 입력하세요.

[STAGE 2] 금고 잠금해제: 진단 기준
DSM-5-TR에 따르면 조현병 진단을 위해 핵심 증상 중 2개 이상이 (A)개월 이상 나타나야 하며, 전체적인 교란의 징후는 최소 (B)개월 이상 지속되어야 합니다.

※ 핵심 증상: 망상, 환각, 와해된 언어

답: (A) + (B) 의 결과값을 숫자로 입력하세요.

[STAGE 3] 증상 분류 카드
환자 이○○ 군은 "방안에 틀어박혀 꼼짝하지 않고(A)", "끼니를 거르며(B)", "나를 비난하는 목소리가 들린다(C)"고 합니다. 아래 보기에서 음성 증상에 해당하는 알파벳의 개수는?

(A) 무의욕(Avolition) (B) 무언어증(Alogia) (C) 환청(Hallucination) (D) 망상(Delusion)

답: 음성 증상인 것의 개수를 숫자로 입력하세요.

[STAGE 4] 약물 수용체 키
2세대 항정신병약(SGA)이 1세대(FGA)와 가장 구별되는 특징은 D2 수용체 차단 외에 이 수용체를 강력하게 차단하여 음성 증상을 개선한다는 점입니다. 이 수용체의 이름은?

답: 영문과 숫자 조합 (예: D2)

[STAGE 5] 부작용 타임라인
부작용이 발생하는 시점이 빠른 순서대로 나열된 번호를 고르세요.

1. Tardive dyskinesia → Akathisia → Acute dystonia
2. Acute dystonia → Akathisia → Tardive dyskinesia
3. Akathisia → Pseudoparkinsonism → Acute dystonia

답: 해당 번호를 입력하세요.

[STAGE 6] 혈액 검사실
Clozapine 투여 시 ANC(절대 중성구 수) 모니터링은 필수입니다. ANC를 계산하는 공식은 다음과 같습니다.

ANC = Total WBC x ( % (B) + % Bands ) / 100

답: (B)에 들어갈 성숙 중성구의 명칭은? (한글 2글자 또는 영문 4글자)

[STAGE 7] 내분비실의 경고
2세대 항정신병약 중에서도 특히 D2 차단력이 강해 고프로락틴혈증을 빈번하게 일으키는 두 가지 약물(성분명)을 고르세요.

1. Quetiapine   2. Risperidone   3. Aripiprazole   4. Paliperidone

답: 두 번호를 합친 숫자를 입력하세요. (예: 1번과 2번이면 3)

[STAGE 8] 처방 알고리듬
국내 가이드라인 3단계에서, 2가지 이상의 약물 시도에도 반응이 없는 치료저항성 환자에게 단일요법으로 권장되는 약물은 무엇일까요?

답: 해당 약물의 성분명을 영문으로 입력하세요.

[STAGE 9] 주사실의 비밀
Paliperidone 주사제인 인베가 서스티나에서 인베가 트린자로 전환할 때, 서스티나 유지 용량의 몇 배를 트린자 용량으로 설정해야 할까요?

답: 숫자만 입력 (소수점 포함)

[STAGE 10] 응급실: NMS의 징후
근육 강직과 고열이 나타나는 NMS 환자의 혈액 검사에서는 근육 세포 파괴로 인해 이 효소 수치가 급격히 상승합니다.

답: 이 효소의 영문 약자 2글자를 입력하세요.
